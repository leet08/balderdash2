What is Balderdash?


Each player receives either an obscure word, movie title, or law. All players must secretly write an ending for each type of question- the word definition, summary of the movie based on its title, or the ending to the law. All the ending choices are shuffled and returned to the group. Players must then vote for the ending they think is authentic.
A player gets 1 point for each vote that their submission receives, and an additional 2 points if they vote for the correct ending.
The player with the most points wins!

How to play?


One person should be designated as Moderator and create a new game by clicking Create New Game. They will receive a Game ID. Each player can then enter the Game ID and create a new username, then click Enter Game (maximum is five players per game). Once the Moderator confirms that all players have joined, they will instruct all players to click the given links to advance to the Writing Round. Following that, players will go into the Voting Round, and finally see the Results.
